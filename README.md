# AfterImage

AfterImage は、表示された数字・文字・絵を短時間で記憶して4択で答えるメモリゲームです。

## セットアップ

```bash
npm install
```

## 開発

`src/` 配下を編集してください。

## ビルド

```bash
npm run build
```

このコマンドで `src/` の内容を `docs/` にコピーします。
GitHub Pages などで `docs/` を公開用ディレクトリとして使えます。

## ディレクトリ構成

- `src/`: 開発用ソース
- `docs/`: 公開用成果物
- `scripts/build-docs.js`: `src/` から `docs/` へコピーするビルドスクリプト
